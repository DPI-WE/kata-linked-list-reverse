# Reverse a Linked List

Write a method that:

- takes the head of a singly linked list
- reverses the list
- returns the new head of the reversed list

## Examples

### List 1 -> 2 -> 3 -> nil

- input: 1 -> 2 -> 3 -> nil
- output: 3 -> 2 -> 1 -> nil

### List 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> nil

- input: 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> nil
- output: 6 -> 5 -> 4 -> 3 -> 2 -> 1 -> nil

### Empty list

- input: nil
- output: nil

### Single element list

- input: 1 -> nil
- output: 1 -> nil
